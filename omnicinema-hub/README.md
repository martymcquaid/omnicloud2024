# OmniCloud Hub

## Project info

OmniCloud Cinema Distribution – web UI for the OmniCloud DCP management system.

## How to run locally

Requirements: Node.js & npm – [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build for production

```sh
npm run build
```
